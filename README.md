# FitLife Club — Лендинг фитнес-клуба

**Тема**: Лендинг фитнес-клуба с тренажёрным залом и расписанием групповых тренировок.

**Фреймворк**: [Bulma](https://bulma.io/)

**Функции**:
- Онлайн-расписание групповых тренировок
- Тарифы с пробным занятием
- WhatsApp-запись
- Полностью адаптивный дизайн
